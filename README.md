# Forcing Mars · 强渡火星

> 在锈红色的星球上，人类的第一个前哨站正等待被建立。而你，是那位必须独自深入火星地下的宇航员。

## 故事背景

二十八世纪末，地球资源几近枯竭。联合星历委员会将最后一批开拓资源投向了火星——那颗在人类仰望了数千年后，终于触手可及的红色星球。

你作为先遣宇航员，降落在一片被风暴侵蚀的古老高原下。地表狂风呼啸、辐射肆虐，根本不适合长期驻留。扫描器却在这片荒原之下探测到稳定的岩层与地热信号：真正的家园，也许不在地表，而在地下。

于是，你穿上宇航服，启动钻探舱，开始向火星深处下潜。

## 下潜深度

### 地表 · 0m

火星的地表是一片死寂的赤色荒原，但生命总会找到它自己的缝隙。幼小的**火星幼蛭**在岩缝中蠕动，贪婪地吸收任何热源；而**沙丘跃行者**则借着风暴的掩护，从沙暴中向你扑来。

这是你面对火星的第一课：这里的空气不能呼吸，这里的敌意却真实可触。

### 地下浅层 · 500m

穿过风化层，地下的温度逐渐稳定，岩壁上开始出现潮湿的锈迹。在这里，你遇见了更顽强的原生生物。**红土爬行者**在狭窄的隧道中游走，懂得在攻击与防御之间切换；而**晶化寄生虫**则像病毒一样适应着你的战术，每一回合都比上一次更具威胁。

每一米下潜，火星都在告诉你：它不会轻易让出这片土地。

### 地核深处 · 2000m

两千米的深处，地热让岩石发出暗红色的光芒。你终于接近了目标区域——适合建立永久前哨站的地下空腔。

但这里也是火星真正的心脏。**地底潜伏者**守护在这片热土之上，厚实的护盾与沉重的打击迫使你倾尽所有。当你终于击败它，地层却开始震颤——扫描器发出刺耳的警报：一个远超预期的巨大生命信号正在苏醒。

**火星吞噬者**从岩浆裂隙中升起。它没有眼睛，却准确无误地锁定了你。它会在两轮蓄力后释放致命一击，那是足以熔化钻探舱的热浪。

这是最后的对决。

## 使命

你不是来征服火星的。你是来为人类寻找一个可以落脚的地方。

每一次出牌，都是一次抉择：是用激光撕裂眼前的威胁，还是用电浆护盾撑过下一回合？电池有限，氧气有限，勇气也一样。

当你最终击败火星吞噬者，警报解除，地热读数稳定，前哨站的坐标被点亮。你会看到一行字：

> **MISSION SUCCESS**  
> 火星安全前哨站已成功建立。

在这片锈红色的星球上，人类终于立住了脚跟。新的家园，正在诞生。
